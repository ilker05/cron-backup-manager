# Cron Backup Manager

## Proje Nedir?
Cron Backup Manager, zamanlanmış görevleri (cron) yöneten, klasör yedeklemesi yapan ve sistem aktivitelerini loglayan modern bir Streamlit arayüzüdür. Bu proje bir okul ödevi olarak tasarlanmıştır.

## Özellikler
- **Cron Translator**: Karmaşık cron ifadelerini insan tarafından okunabilir metne dönüştürür.
- **Backup Wizard**: Kaynak klasörleri zip formatında hedef klasöre yedekler.
- **Log Viewer**: Tüm yedekleme ve sistem işlemlerini gerçek zamanlı olarak görüntüler.
- **Self-Check**: Yedekleme sonrası dosya bütünlüğünü otomatik kontrol eder.

## Kurulum
Projeyi çalıştırmak için gerekli kütüphaneleri yükleyin:
```bash
pip install -r requirements.txt
```

## Çalıştırma
Arayüzü başlatmak için:
```bash
streamlit run src/app.py
```

## Klasör Yapısı
- `researchs/`: Araştırma notları.
- `specs/`: Teknik gereksinimler.
- `src/`: Kaynak kodlar.
- `docs/`: Dokümantasyon.
